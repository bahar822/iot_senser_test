# ドットマトリックス

## 環境準備

下記ライブラリをArduino IDEに追加する

* https://github.com/markruys/arduino-Max72xxPanel
* https://github.com/adafruit/Adafruit-GFX-Library


## 配線

配線は下記の通り

| DotMatrix | ESP32 |
| --------- | ----- |
| VCC       | 5V    |
| GND       | GND   |
| DIN       | 23    |
| CS        | 15    |
| CLK       | 18    |
